Basics of the Web

HTML = Hypertext markup Language (think “documents”)
Major pieces: You, using computer browser to and from the Internet uses HTML to and from Servers

HTML is made up of
• Text content (what you see)
• Mark up (what it looks like)
• References to other docs eg images and videos
• Links to other pages

HTML Mark Up Vocabulary

Tag - Term used by HTML. Tags are always contained within angled brackets. Some tags have both an opening and closing bracket, but some are called “void” tags and only have the opening brackets

Element - this term is used to describe everything within opening and closing tags which looks like <tag>content</tag>

Attribute - is a a property of an HTML element. For example, href is the atribute used to display a URL. For example use this to display a URL <a href=www.udacity.com;>derp</a>
